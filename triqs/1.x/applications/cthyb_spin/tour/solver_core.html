Options and accessors¶

The solver consists of one main object with a constructor and a solve() method. Both have a series of options, some mandatory, some optional. They are summarized below.

The solver object also comes with a number of accessors in order to set the inputs of the solver, as well as access its outputs. The accessors are summarized below.
