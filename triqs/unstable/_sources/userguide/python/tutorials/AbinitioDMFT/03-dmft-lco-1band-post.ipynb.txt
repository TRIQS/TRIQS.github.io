    "## 🌀 Analytic continuation: from Matsubara to real frequencies"
   ]

   "source": [
    "In this section, we will learn how to **analytically continue** our results from Matsubara (imaginary) frequencies onto the real-frequency axis, so that we may plot spectral functions. Because analytic continuation is an ill-posed problem, it is notoriously unstable, and the method itself remains an active area of research. However, there are two common analytic continuation techniques: Padé analytic continuation and the maximum entropy (MaxEnt) method. In this section, we will use Padé analytic continuation since it is simpler. We will then compare our results with MaxEnt.\n",
    "\n",
    "### ❓The problem\n",
    "\n",
    "We assume the Matsubara Green's function $G(i\\omega_{n})$ corresponds to a dynamic correlation function $G(\\omega)$, where $\\omega_{n}$ is the Matsubara frequency, and $\\omega$ is the real frequency. The function $G(z)$ is analytic in the whole upper complex plane of $z$ excluding the real axis. The retarded Green's function can be obtained by analytic continuation: $$G(\\omega) = G(i\\omega_{n}\\rightarrow \\omega + i0^{+}).$$ The spectral function $A(\\omega)$ is determined by taking the imaginary part of $G(\\omega + i0^{+})$ using the formula\n",
    "\n",
    "$$ A(\\omega) = -\\frac{1}{\\pi}\\mathrm{Im}G(\\omega + i0^{+}).$$\n",
    "\n",
    "### 📐 Padé method\n",
    "\n",
    "Padé analytic continuation is based on the assumption that the Matsubara Green's function can be approximated by a rational function of degree $r$:\n",
    "\n",
    "$$ G(z) = \\frac{\\sum_{i=0}^{r-1} p_{i}z^{i}}{\\sum_{j=0}^{r-1}q_{j}z^{j} + z^{r}}, $$\n",
    "\n",
    "where $p_{i}$ and $q_{j}$ are complex. The coefficients can be determined by solving $2r$ linear equations from $2r$ arbitrary but different input points ${i\\omega_{n}, G(i\\omega_{n})}$. Once the coefficients are determined, we replace $z$ with $\\omega + i0^{+}$ to obtain the retarded correlation function."

    "As you can see, the two analytic continuation methods, Padé and MaxEnt, yield different results that share only some qualitative features. MaxEnt is generally more robust to numerical noise and produces smooth spectral functions, making it a reliable choice when the input data are noisy, as is often the case in quantum Monte Carlo simulations. Padé approximants, by contrast, can capture fine structures with the high accuracy when provided with very precise data, but they are highly sensitive to noise as you've observed.\n",
    "\n",
    "In practice, MaxEnt is typically preferred for stochastic data, while Padé is more useful when working with high-precision input or for cross-checking features suggested by MaxEnt. Both methods remain viable and can produce high-quality spectra when applied in the appropriate context."
