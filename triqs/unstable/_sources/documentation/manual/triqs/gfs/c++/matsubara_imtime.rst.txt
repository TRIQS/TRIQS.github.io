gf_mesh<imtime>
=================

**Synopsis**:

.. code-block:: c

      class gf_mesh<imtime>;

Imaginary-time mesh

  Mesh for the imaginary-time axis from :math:`0` to :math:`\beta`.
  The time points are defined as follows, for :math:`n=0{\dots}N-1`:

.. math::
		\tau_n=\frac{n}{N-1}\beta

..
 where :math:`N` is the size of the mesh.
