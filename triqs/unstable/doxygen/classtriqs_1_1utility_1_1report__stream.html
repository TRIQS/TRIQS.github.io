triqs::utility::report_stream

#include <triqs/utility/report_stream.hpp>

Detailed Description

Output stream with flexible verbosity level.

This class behaves pretty much like a standard ostream but you can also set some verbosity level.

Example: 

report_stream rep(std::cout, 2); // verbosity level 2
rep << "Hello" << endl;          // prints Hello
rep(2) << "Hello2" << endl;      // prints Hello2
rep(3) << "Hello3" << endl;      // verbosity < 3, no output
