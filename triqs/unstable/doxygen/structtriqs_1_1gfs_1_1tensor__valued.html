Detailed Description

template<int R>
struct triqs::gfs::tensor_valued< R >

Target type for a complex tensor-valued Green's function.

The value stored at each mesh point is a complex tensor of rank \( R \), i.e. an nda array of rank \(R \) with std::complex<double> value type.

Template Parameters
RRank of the tensor (must be strictly positive).
