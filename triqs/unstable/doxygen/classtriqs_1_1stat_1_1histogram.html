triqs::stat::histogram

#include <triqs/stat/histograms.hpp>

Detailed Description

Statistical histogram.

This class serves to sample a continuous random variable, and to 'bin' it. It divides a given range of real values into a series of equal intervals, and counts amounts of samples falling into each interval. The histogram keeps track of the total number of the sampled values, as well as of the lost samples that lie outside the chosen range.

◆ operator<<()

histogram & triqs::stat::histogram::operator<< ( double x)

Bin a real value into the histogram.

Bins a real value into the histogram. A sampled value :math:x falls into the :math:n-th bin if :math:x\in[nh-h/2;nh+h/2[, where :math:h is the bin length, :math:h = (b - a) / (n_\mathrm{bins} - 1). This convention implies that the first (:math:n = 0) and the last (:math:n = n_\mathrm{bins} - 1) bins are effectively two times shorter that others. Sample is discarded if :math:x\notin[a; b].

Parameters
xSampled value
Returns
Reference to *this, so that it is possible to chain operator<< calls
