Manipulating local Green’s functions¶

Let us start with a problem of free electrons: an impurity d level embedded in a flat conduction bath \(\Delta\) of s-electrons. To construct and plot the corresponding 2x2 Green’s function:

\[\begin{split}\hat{G}^\mathrm{s+d} ( \omega) = \begin{pmatrix} \omega - \epsilon_d & t \\ t & \Delta^{-1} \end{pmatrix}^{-1}\end{split}\]

we first create the corresponding energy mesh on the real axis in the energy interval \(\omega \in [-2,2]\). The Green’s function is generated using classes of the gf module by setting up \(\left[\hat{G}^\mathrm{s+d}\right]^{-1}\) and inverting it. Finally, the obtained bath and impurity densities of states are plotted using the TRIQS function oplot:

from pytriqs.gf import GfReFreq, Omega, Wilson, inverse
import numpy

eps_d,t  = 0.3, 0.2

# Create the real-frequency Green's function and initialize it
g = GfReFreq(indices = ['s','d'], window = (-2, 2), n_points = 1000, name = "$G_\mathrm{s+d}$")
g['d','d'] = Omega - eps_d
g['d','s'] = t
g['s','d'] = t
g['s','s'] = inverse( Wilson(1.0) )
g.invert()

# Plot it with matplotlib. 'S' means: spectral function ( -1/pi Imag (g) )
from pytriqs.plot.mpl_interface import oplot, plt
oplot( g['d','d'], '-o', RI = 'S', x_window  = (-1.8,1.8), name = "Impurity" )
oplot( g['s','s'], '-x', RI = 'S', x_window  = (-1.8,1.8), name = "Bath" )
plt.show()
