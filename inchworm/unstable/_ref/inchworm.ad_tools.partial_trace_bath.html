inchworm.ad_tools.partial_trace_bath

inchworm.ad_tools.partial_trace_bath()

Dispatched C++ function(s).

[1] (ad_tot: AtomDiagReal,
     ad_target: AtomDiagReal,
     ad_bath: AtomDiagReal,
     beta: float,
     tau: float)
  -> ndarray[ndarray[float, 2], 1]

Calculate U(tau) = Tr_bath [ exp[-(beta - tau) * H_bath] * exp(-tau * H_tot) ] / Z_bath

Within atom_diag, perform partial trace over indices above nfops_imp, i.e. the bath degrees of freedom. Only the nfops_imp first degrees of freedom will be preserved.

Parameters:
ad_totAtomDiagReal

atom_diag of the full Hamiltonian H = H_loc + H_bath + H_hyb.

ad_imp

atom_diag of the local Hamiltonian H_loc.

ad_bathAtomDiagReal

atom_diag of the bath Hamiltonian H_bath (in the full basis).

betafloat

inverse temperature.

taufloat

0 < tau < beta.

Returns:
ndarray[ndarray[float, 2], 1]

The partial sum matrix of a function H. The result is a block diagonal matrix, with blocks and indices in the same order as in ad_imp.
