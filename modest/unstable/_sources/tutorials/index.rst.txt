.. _tutorials:

Tutorials
*********

.. note::

   This page is a work in progress. Native ModEST tutorials will be added in
   future releases.

In the meantime, we recommend the ``AbinitioDMFT`` tutorial series in the
upstream `TRIQS/tutorials <https://github.com/TRIQS/tutorials>`_ repository
as a starting point. It walks through end-to-end DFT+DMFT calculations on
real materials (La\ :sub:`2`\ CuO\ :sub:`4`, SrVO\ :sub:`3`) using
ModEST and the broader TRIQS ecosystem.
