C++ API
*******

The C++ layer of ModEST (``c++/triqs_modest/``, namespace
``triqs::modest``) is documented with Doxygen. The Python interface is a
generated wrapper around this library, so the C++ reference is the primary
source of truth for type signatures, template parameters, and the internals
of every routine.

`Open the C++ reference manual (Doxygen). <../doxygen/index.html>`_
