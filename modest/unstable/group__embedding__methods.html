Embedding

Detailed Description

The embedding methods are functions that can be used to manipulate the block structures of matrices, tensors, or TRIQS Green's functions. The two methods are: (1) embed and (2) extact.

Embed (embedding) maps a list of objects from the impurity to the full "embedded space".

Extract (extracting) maps the object from the "embedded space" to a list of unique impurities that can be solved with an impurity solver.
