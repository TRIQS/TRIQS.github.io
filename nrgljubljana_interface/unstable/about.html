About nrgljubljana_interface¶

nrgljubljana_interface provides a C++ and Python interfaces between “NRG Ljubljana” numerical renormalization group code and TRIQS. It is based on problem template files and does not require Mathematica during computation (but Mathematica is required for generating new template files for custom Hamiltonians and symmetry types).

Written and maintained by R. Žitko and N. Wentzell.
