Nevanlinna
*********

.. sidebar:: Nevanlinna |PROJECT_VERSION|

   This is the homepage of Nevanlinna |PROJECT_VERSION|.
   For changes see the :ref:`changelog page <changelog>`.

      .. image:: _static/logo_github.png
         :width: 75%
         :align: center
         :target: https://github.com/triqs/Nevanlinna


Nevanlinna analytic continuation is TRIQS application, providing an efficient implementation of the Nevanlinna analytic continuation method for Green's functions `Phys. Rev. Lett. 126 (2021)<https://doi.org/10.1103/PhysRevLett.126.056402>_` and its extension to matrix valued Green's functions `Phys. Rev. B 104 (2021) <https://doi.org/10.1103/PhysRevB.104.165111>`_. The method is implemented in C++ and interfaced to Python based on the `TRIQS software library<https://doi.org/10.1016/j.cpc.2015.04.023>`_ and described in detail in `Comput. Phys. Commun. 304 (2024) <https://doi.org/10.1016/j.cpc.2024.109299>`_.

TRIQS/Nevanlinna is free software distributed under the GPL license.

Learn how to use Nevanlinna in the :ref:`documentation`.
