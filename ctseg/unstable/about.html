Authors¶

The continuous-time hybridization-expansion segment solver has been written by T. Ayral, H. Hafermann, P.Delange, M. Ferrero and O. Parcollet.

Useful references¶

This implementation is based on a hybridization expansion of the partition function as described in reference [1], with dynamical interactions in the charge and longitudinal spin channel (as described in [2]) as well as transverse spin channel (as described in [3]). The computation of the one-particle Green’s function has been improved with the use of Legendre polynomials following reference [4]. The various improved estimators implemented in the code follow [5] and [6]. The code is based on the TRIQS toolbox [7].
