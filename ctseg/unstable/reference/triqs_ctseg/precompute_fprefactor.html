triqs_ctseg::precompute_fprefactor¶

#include <triqs_ctseg.hpp>

class precompute_fprefactor

Object which precomputes the prefactor \(I_{o_1}(\tau)\) for the improved estimator

\(I_{o_1}\) is used in the computation of the improved estimator \(F_{o_1}(\tau)\). It is defined as:

\[I_{o_1}(\tau) = \int_0^\beta d\bar{\tau} \sum_{o_2}\]

hat{mathcal{U}}_{o_1,o_2}(tau-bar{tau}) n_{o_2}(bar{tau})

models the concept of mc_generic auxiliary precomputation.

Note

In the measurement all times are accessed for a given orbital and not

vice versa; Storage in vector of maps instead of map of vectors keeps the individual maps small
