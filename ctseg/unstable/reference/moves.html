Monte-Carlo updates (moves)¶

The Monte-Carlo space of configurations is explored using “updates” or “moves”. In this page, we list the updates implemented in the code. All updates can be turned on and off using the options of the solve() method of the solver (options corresponding to moves all start with move_ by convention).

General updates¶

The last two moves enhance, but are in general not necessary for, ergodicity.

Updates for dynamical transverse spin interactions¶

The last two moves enhance, but are in general not necessary for, ergodicity; they are an experimental feature and may still contain bugs.
