hartree_fock.impurity.ImpuritySolver¶

class hartree_fock.impurity.ImpuritySolver(gf_struct, beta, n_iw=1025, symmetries=[], force_real=False)[source]¶

Hartree-Fock Impurity solver.

Parameters:

gf_struct : list of pairs [ (str,int), …]

Structure of the Green’s functions. It must be a list of pairs, each containing the name of the Green’s function block as a string and the size of that block. For example: [ ('up', 3), ('down', 3) ].

beta : float

inverse temperature

n_iw: integer, optional. :

Number of matsubara frequencies in the Matsubara Green’s function. Default is 1025.

symmeties : optional, list of functions

symmetry functions acting on self energy at each consistent step

force_real : optional, bool

True if the self energy is forced to be real

Methods

__init__(gf_struct, beta[, n_iw, …]) Initialize self.
interaction_energy() Calculate the interaction energy
solve(h_int[, with_fock, one_shot]) Solve for the Hartree Fock self energy using a root finder method.
