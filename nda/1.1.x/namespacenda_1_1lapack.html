LU decomposition of a matrix

The matrix m is modified during the operation. The matrix is interpreted as FORTRAN ordered, and then LU decomposed. NB : for some operation, like det, inversion, it is fine to be transposed, for some it may not be ...

Computes the solution to the system of linear equations with a tridiagonal coefficient matrix A and multiple right-hand sides.

The routine solves for X the system of linear equations A*X = B, where A is an n-by-n tridiagonal matrix. The columns of matrix B are individual right-hand sides, and the columns of X are the corresponding solutions.
