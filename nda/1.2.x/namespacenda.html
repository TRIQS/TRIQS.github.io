◆ get_block_layout()

template<MemoryArray A>
std::optional< std::tuple< int, int, int > > nda::get_block_layout ( A const &  a)

If the array is block-strided, return the number of blocks, their size and the stride

An array is block-strided if its data in memory is laid out as contiguous blocks of the same size repeated with a single stride in memory

We check this by asserting that (s=strides, l=lengths)

  • there is at most one strided index m with s_m != s_{m+1}*l_{m+1}
  • all other indices are 'contiguous': s_i = s_{i+1}*l_{i+1}

Example of a block-strided layout with n_blocks=4, block_size=4 and block_stride=6 (x: data, _: other memory): xxxx__xxxx__xxxx__xxxx__
