atom_diag¶

Synopsis:

   template<bool Complex> class atom_diag;

This class is provided as a simple tool to diagonalize Hamiltonians of
finite fermionic systems of a moderate size.

Template parameters¶

  • Complex: Allow the Hamiltonian to be complex.
