Analytic continuation with OmegaMaxEnt_TRIQS¶

OmegaMaxEnt_TRIQS is a python TRIQS interface to the maximum entropy analytic continuation program \(\Omega MaxEnt\) written in C++ (program’s main page). It also automates the analytic continuation of matrix-valued Green’s functions and therefore extends the functionality of \(\Omega MaxEnt\) using TRIQS.

The present documentation contains installation instructions, a user guide, and tutorials.

See also the About OmegaMaxEnt_TRIQS page for citations.
