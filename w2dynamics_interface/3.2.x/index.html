The w2dynamics interface

w2dynamics_interface 3.0.0

This is the homepage of w2dynamics_interface v3.0.0. For changes see the changelog page.

w2dynamics_interface is a python interface of the Würzburg / Wien strong coupling impurity solver w2dynamics.

The present documentation contains installation instructions.

See also the About w2dynamics_interface page for citations.

Learn how to use w2dynamics_interface in the Documentation.
